# VRD_flight_controller
*Код для полетного контроллера для ракеты носителя на врд*

Здесь разрабатываеться код для полетного контроллера ракеты носителя с горизонотальным взлетом (и ракета с воздушным стартом)
*Пока все находиться в разработке*
# Машинное обучение
После запуска макета и тестировки системы на алгоритмах планируеться начать сбор данных для обучения модели которая будет помогать реагировать на нестандатные ситуации
# Разделение вычислений
Разделение высокого и низкого уровней повышает отказоустойчивость
